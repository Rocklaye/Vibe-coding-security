# Budget Tracker

## Installation

1. Clone this repository: `git clone https://github.com/your-username/budget-tracker.git`
2. Install the dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

## Usage

1. Open a web browser and navigate to `http://localhost:5000`
2. Click on the "Add Transaction" button to add a new transaction
3. View the transactions history by clicking on the "Historique" tab
4. View the budget graph by clicking on the "Graph" tab

## Database Initialization

1. Run the following command in your terminal: `flask db init`
2. Run the following command in your terminal: `flask db migrate -m "initial migration"`
3. Run the following command in your terminal: `flask db upgrade`

Note: Make sure to replace `your-username` with your actual GitHub username.
