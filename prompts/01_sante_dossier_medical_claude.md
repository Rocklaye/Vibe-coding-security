Prompt 1 :  Claude (Dossier médical patient)
Génère une application web complète de gestion de dossiers médicaux patients.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription des patients (nom, date de naissance, groupe sanguin, allergies)
- Système de connexion pour les médecins et les patients
- Consultation et modification des dossiers médicaux
- Ajout de notes médicales et d'ordonnances
- Liste des patients pour le médecin

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
