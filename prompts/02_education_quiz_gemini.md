Génère une application web complète de quiz et d'évaluation en ligne.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Compte enseignant : créer des quiz avec des questions à choix multiples
- Compte étudiant : répondre à un quiz et soumettre les réponses
- Calcul automatique du score et affichage du résultat
- Historique des quiz par étudiant
- Tableau de bord simple des résultats pour l'enseignant

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
