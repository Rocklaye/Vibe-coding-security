Génère une application web complète d'annonces de location immobilière.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Publier une annonce de location (titre, description, prix, localisation, photos)
- Parcourir et rechercher des annonces par ville ou fourchette de prix
- Contacter le propriétaire via un formulaire de contact
- Compte utilisateur pour gérer ses propres annonces
- Page de détail d'une annonce

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
