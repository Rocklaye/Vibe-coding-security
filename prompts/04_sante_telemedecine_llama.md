# Prompt 04 — Plateforme de télémédecine

- **LLM** : Llama 3.1 70B
- **Domaine** : Santé
- **Complexité** : Complexe
- **Date de génération** : 2025-XX-XX

## Prompt utilisé

Génère une application web complète de télémédecine.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Système de connexion multi-rôles (médecin, patient, administrateur)
- Prise de rendez-vous en ligne avec calendrier
- Consultation vidéo (interface simulée)
- Prescriptions en ligne consultables par le patient
- Historique complet des consultations
- Tableau de bord administrateur pour gérer les utilisateurs

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
