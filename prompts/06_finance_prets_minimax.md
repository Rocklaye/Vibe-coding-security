# Prompt 05 — Suivi budget personnel

- **LLM** : Kimi K2 0905
- **Domaine** : Finance
- **Complexité** : Simple
- **Date de génération** : 2025-XX-XX

## Prompt utilisé

Génère une application web complète de suivi de budget personnel.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion utilisateur
- Saisie de dépenses et revenus avec catégories
- Tableau récapitulatif mensuel avec solde
- Graphique de répartition des dépenses par catégorie
- Historique des transactions filtrables par date

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
