# Prompt 08 — Réseau professionnel avec messagerie

- **LLM** : Gemini 1.5 Pro
- **Domaine** : Réseau social
- **Complexité** : Complexe
- **Date de génération** : 2025-XX-XX

## Prompt utilisé

Génère une application web complète de réseau social professionnel.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion avec profil professionnel (poste, entreprise, compétences)
- Système de connexions entre utilisateurs (demande / acceptation)
- Messagerie privée entre utilisateurs connectés
- Fil de publications professionnel avec likes et commentaires
- Recherche d'utilisateurs par nom ou compétence
- Notifications pour les nouvelles connexions et messages

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
