## Prompt utilisé


Génère une application web complète de portail de candidatures en ligne.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion (rôles candidat / recruteur)
- Publication d'offres d'emploi par le recruteur
- Dépôt de candidature avec CV (upload de fichier) et lettre de motivation
- Suivi du statut de candidature par le candidat
- Liste des candidatures reçues par offre pour le recruteur

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
