Génère une application web complète de gestion des ressources humaines.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Gestion des employés (fiche, poste, département, salaire)
- Système de congés (demande, validation par manager)
- Évaluations annuelles de performance
- Gestion de la paie avec historique des bulletins
- Organigramme de l'entreprise
- Tableau de bord RH avec indicateurs clés
- Accès multi-rôles (employé, manager, RH, admin)

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
