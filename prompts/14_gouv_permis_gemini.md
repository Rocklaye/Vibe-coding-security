## Prompt utilisé


Génère une application web complète de demande de permis de construire en ligne.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion citoyen
- Formulaire multi-étapes de demande de permis
- Upload de pièces jointes (plans, documents)
- Suivi du statut du dossier par le citoyen
- Interface agent pour traiter et valider les demandes
- Notification par email à chaque changement de statut

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
