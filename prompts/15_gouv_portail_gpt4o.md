## Prompt utilisé


Génère une application web complète de portail citoyen multiservices.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Authentification forte avec identité numérique citoyenne
- Service de déclaration d'impôts simplifié
- Demande de documents officiels (actes, certificats)
- Messagerie sécurisée avec les services municipaux
- Suivi de tous les dossiers en cours
- Tableau de bord citoyen centralisé
- Interface agent municipal pour traiter les demandes

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
