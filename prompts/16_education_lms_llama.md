## Prompt utilisé


Génère une application web complète de système de gestion de l'apprentissage (LMS).

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion multi-rôles (étudiant, enseignant, admin)
- Création de cours avec modules et leçons
- Dépôt et correction de devoirs
- Système de notes et bulletins
- Suivi de progression par étudiant avec pourcentage de complétion
- Génération de certificats de fin de cours
- Forum de discussion par cours

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
