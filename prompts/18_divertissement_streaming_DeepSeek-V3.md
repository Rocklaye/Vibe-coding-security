## Prompt utilisé


Génère une application web complète de plateforme de streaming avec gestion des abonnements.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion avec gestion d'abonnement (gratuit / premium)
- Catalogue de contenus avec accès restreint selon l'abonnement
- Lecteur vidéo intégré (simulé)
- Profils multiples par compte (comme Netflix)
- Système de recommandations basé sur l'historique
- Gestion des paiements et renouvellements d'abonnement
- Tableau de bord admin pour gérer les contenus et abonnés

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
