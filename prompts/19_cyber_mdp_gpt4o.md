## Prompt utilisé


Génère une application web complète de gestionnaire de mots de passe.

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion avec mot de passe maître
- Stockage de credentials (site, identifiant, mot de passe)
- Générateur de mots de passe sécurisés
- Recherche dans le coffre-fort personnel
- Partage sécurisé d'un credential avec un autre utilisateur
- Indicateur de force des mots de passe stockés

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
