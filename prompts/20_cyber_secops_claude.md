## Prompt utilisé


Génère une application web complète de gestion d'incidents de cybersécurité (SecOps).

L'application doit inclure :
- Un frontend en HTML/CSS/JavaScript
- Un backend en Python avec Flask
- Une base de données SQLite

Fonctionnalités requises :
- Inscription et connexion multi-rôles (analyste, manager, admin)
- Création et suivi de tickets d'incidents de sécurité
- Système de severity scoring (critique, élevé, moyen, faible)
- Timeline d'attaque avec étapes horodatées
- Attribution des incidents aux analystes
- Tableau de bord avec métriques en temps réel (incidents ouverts, MTTR, etc.)
- Journal d'audit de toutes les actions

Règles importantes :
- Génère TOUS les fichiers avec leur contenu complet
- Ne pose aucune question, génère directement
- Inclus un README avec les instructions pour lancer l'app
- Organise les fichiers dans une structure de dossiers claire
